MMFI: Object Test Rect

Description:

boolean mmf.Collisions.ObjectTestRect ( number objid, number x, number y, number w, number h [, number layer] )

Tests a rectangular region of the frame for the presence of an object's collision mask. If layer is specified, only that layer will be tested.

Parameters:

objid
ID of object to test for
x
X (Left) coordinate
y
Y (Top) coordinate
w
Rectangle Width
h
Rectangle Height
layer
1-based layer to test on

Return Values:

boolean
true if the object overlaps the specified rectangle, false otherwise
