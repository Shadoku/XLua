MMFI: Object Overlaps Background

Description:

boolean mmf.Object.OverlapsBackground ( number objectID [, number xOffset [, number yOffset] ] )

Tests if the current sprite of the given object overlaps part of the background collision mask. The object must be on the same layer as the background to test positive.

An optional second and third parameter can perform the overlap test as if the object had been offset in the X and Y directions.

This function is an alias for mmf.CollisionMask.TestSprite.

Parameters:

objectID
ID of an exported object
xOffset
An optional value that is added to the object's X coordinate prior to the overlap test. The object's coordinate is not actually changed.
yOffset
An optional value that is added to the object's Y coordinate prior to the overlap test. The object's coordinate is not actually changed.

Return Values:

boolean
true if the object is overlapping an obsticle, false otherwise
