MMFI: Export Object

Description:

void mmf.Object.Export ( number fixedId, number objectID )

Exports an object to the MMF Interface from an object's fixed value. Some functions return fixed values of objects that may not be exported, and this function allows them to be exported internally.

On successful export, this function will call mmf.ExportCallback if it is defined.

Parameters:

fixedId
Fixed ID of an object in MMF.
objectId
ID to assign the object.

Return Values:

none
