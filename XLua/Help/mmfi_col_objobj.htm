MMFI: Object Test Object

Description:

boolean mmf.Collisions.ObjectTestObject ( number objid, number objid2, number objid2 [, number layer [, number xoff [, number yoff] ] ] )

Tests if the collision masks of two different objects are intersecting. If layer is specified, only that layer will be tested (the first object must exist on that layer). If an X-offset or Y-offset are specified, they will be added to the second object's coordinates before testing.

The layer parameter may additionally be either of the special constants mmf.SAME_LAYER or mmf.DIFF_LAYER.

Parameters:

objid
ID of first object
objid2
ID of second object
layer
1-based layer to test on
xoff
Distance in pixels to offset second object's mask on the X axis
yoff
Distance in pixels to offset second object's mask on the Y axis

Return Values:

boolean
true if the two objects intersect (overlap), false otherwise
