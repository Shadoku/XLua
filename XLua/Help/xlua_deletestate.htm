XLua: Delete State

Description:

boolean xlua.DeleteState ( number stateID )

Attempts to delete an existing state, if the calling state has been granted the REMOTE_DESTROY permission in the target state.

Even if a state has the correct permissions, the target state cannot be deleted if it is currently bound to an XLua object in any frame.

Parameters:

stateID
ID of the state to delete

Return Values:

boolean
true on Success, false otherwise
