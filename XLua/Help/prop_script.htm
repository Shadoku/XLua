Properties ‣ Script

These properties deal with the Lua script

Embedded Script

Can be used to define a single Lua script of arbitrary length that is stored inside the XLua object, rather than in another object or an external file.

Run Script At Start

If checked, the embedded script will be executed at the start of the frame, after other XLua initialization has occured and other properties have taken effect. If Bind State is uncheked, this option is ignored.

Load Standard Libs

Loads all of Lua's standard libraries, including the bitop library, into the bound state. If Bind State is uncheked, this option is ignored.

Auto-Register Functions

Automatically registers any function named in an "On Function" condition at the start of the frame, and prior to executing the embedded script if that option is selected. All functions are registered without a minimum parameter requirement. If Bind State is unchecked, this option is ignored.
