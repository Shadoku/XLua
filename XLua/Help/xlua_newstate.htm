XLua: New State

Description:

boolean xlua.NewState ( number stateID )

Creates a new, empty Lua state that is not currently bound to any XLua object. If the specified stateID is already in use, this function will fail.

On success, the calling state will have REMOTE_LOAD and REMOTE_DESTROY permissions in the newly created state (esentially, the calling state owns the new state).

Parameters:

stateID
ID to associate with the new State

Return Values:

boolean
true on Success, false otherwise
