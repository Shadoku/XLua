Properties ‣ Interfaces

These properties deal with optional built-in interfaces to external components.

Use MMF Interface

When checked, the MMF Interface is enabled for the bound state, and the mmf table is loaded. This option is ignored if the Bind State property is unchecked.

Use WIN Interface

When checked, the WIN Interface is enabled for the bound state, and the win table is loaded. This option is ignored if the Bind State property is unchecked.
