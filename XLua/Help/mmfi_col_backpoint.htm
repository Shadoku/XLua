MMFI: Backdrop Test Point

Description:

boolean mmf.Collisions.BackdropTestPoint ( number x, number y [, number layer] )

Tests a single pixel of the background collision mask. If layer is specified, only that layer will be tested.

Parameters:

x
X coordinate
y
Y coordinate
layer
1-based layer to test on

Return Values:

boolean
true if there is an obsticle, false otherwise
