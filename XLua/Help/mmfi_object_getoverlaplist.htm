MMFI: Get Object Overlap List

Description:

table mmf.Object.GetOverlapList ( number objectID, number classID [, number layerMode] )

Tests if the current object is overlapping any object that is a member of the object class, and returns the fixed ID of each overlapping instance. An object class is the unique identifier that is shared by all instances of the same object. See mmf.Object.GetClass.

Parameters:

objectID
ID of an exported object
classID
ID of an object class. If this value is set to the special value mmf.ANY_CLASS, every overlapping object will be returned.
layerMode
An optional value that takes one of two constants from the mmf namespace: mmf.SAME_LAYER or mmf.ANY_LAYER. If mmf.ANY_LAYER is specified, the overlap test will ignore layers. If mmf.SAME_LAYER is specified, the test will fail if the two objects are on separate layers. mmf.SAME_LAYER is the default value.

Return Values:

table
Table containing the fixed ID of each overlapping object.
