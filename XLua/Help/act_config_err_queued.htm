Actions ‣ Config ‣ Error Event Mode
Set Queued

Sets future error events to queued mode. Queued mode causes any On Error triggered events from when errors occur to be queued up and executed at the end of MMF's event loop.

Queued mode execution is safe, but limits in the size of MMF's event queue may cause other queued events from XLua or other objects to get dropped if too many events are triggered in the same event loop. Immediate mode can get around this problem.

Parameters

None.
