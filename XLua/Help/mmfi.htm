MMF Interface

The MMF Interface is a collection of functions used inside your lua code to directly access or set internal state in your objects or application. The interface must be explicitly enabled by the appropriate action, otherwise these functions will be unavailable.

Note: By using this interface, you may break your application's compatibility with future versions of Multimedia Fusion! Setting values also caries some degree of risk. Use with care.

The entire MMF Interface is layed out in the global table mmf. Sets of functions may be grouped in subtables, such as Object.
