MMFI: Get Objects Over Point

Description:

table mmf.Collisions.GetObjectsOverPoint ( number x, number y [, number classid [, number layer] ] )

Returns a table containing the fixed IDs of all objects overlapping a given point in the frame. If a class ID is specified, only objects belonging to that class will be returned. If layer is specified, only that layer will be tested.

To get an object's class ID, use mmf.Object.GetClass.

To get an object from a fixed ID, use mmf.Object.Export.

Parameters:

x
X coordinate
y
Y coordinate
classid
ID of object class to test for
layer
1-based layer to test on

Return Values:

boolean
table of fixed IDs
