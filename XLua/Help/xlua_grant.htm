XLua: Grant

Description:

void xlua.Grant ( number feature, number to_states... )

Grants permission to a remote state to exercise the specified "feature" in the current state. Currently, the available features are the following enumerations:

  • xlua.REMOTE_LOAD: Allows a remote state to load and execute Lua code in this state.
  • xlua.REMOTE_DESTROY: Allows a remote state to destroy this state.

The REMOTE_LOAD permission gives remote states virtually free reign over the current state and should be used with care. Even if the REMOTE_DESTROY permission is granted, no state can be destroyed while it is still bound to an XLua object.

Parameters:

feature
A valid permission enumeration
to_states
One or more states to grant this permission to

Return Values:

None.
