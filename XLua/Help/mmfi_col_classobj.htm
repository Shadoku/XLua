MMFI: Object Class Test Object

Description:

boolean mmf.Collisions.ObjectClassTestObject ( number classid, number objid, number objid2 [, number layer [, number xoff [, number yoff] ] ] )

Tests if any object of an object class is overlapping a specific object. If layer is specified, only that layer will be tested. If an X-offset or Y-offset are specified, they will be added to the second object's coordinates before testing.

The layer parameter may additionally be either of the special constants mmf.SAME_LAYER or mmf.DIFF_LAYER.

To get an object's class ID, use mmf.Object.GetClass.

Parameters:

classid
ID of object class to test for
objid
ID of object to use as collision mask
layer
1-based layer to test on
xoff
Distance in pixels to offset object's mask on the X axis
yoff
Distance in pixels to offset object's mask on the Y axis

Return Values:

boolean
true if an object of the specified class overlaps the specified object, false otherwise
