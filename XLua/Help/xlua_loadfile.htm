XLua: Load File

Description:

boolean xlua.LoadFile ( number stateID, string file )

Loads a Lua script into a remote state and executes it. If the call succeeds, it will return true.

A state can only load Lua code into a target state if the target state has granted the REMOTE_LOAD permission to the calling state.

Parameters:

stateID
The remote state in which to load Lua code
name
Filename of Lua script to load

Return Values:

boolean
true if the call succeeds, false otherwise
