MMFI: Test Collision Mask Rectangle

Description:

boolean mmf.CollisionMask.TestRect ( number x, number y, number sizex, number sizey, number layer )

Tests a rectangular area of the background collision mask on the specified layer for an obsticle.

Parameters:

x
X coordinate of top-left corner of rect
y
Y coordinate of top-left corner of rect
sizex
Width of rect
sizey
Height of rect
layer
0-based layer to test on

Return Values:

boolean
true if there is an obsticle in any part of the rect, false otherwise
