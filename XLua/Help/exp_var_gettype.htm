Get Variable Type
Function: VarType$
Returns: String

Returns the type of a variable in Lua. The most common types that will be returned are nil, boolean, number, string, table, and function. Other valid types that may be returned are userdata, thread, proto, and upval.

If the requested variable does not exist, the return type is nil. If the XLua object is currently unbound, the return type is the empty string. Click here for more information on valid naming.
