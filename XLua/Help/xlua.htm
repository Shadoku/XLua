XLua Interface

The XLua Interface is a library for directly manipulating XLua states. This library allows such tasks as creating and deleting states, executing code across states, or calling functions across states. Build in access control prevents states from manipulating each other without permission first being granted. These facilities would make it possible to, for example, create a separate state that acts as a sandbox for untrusted code.

The XLua interface is available in the global table xlua.

Quick Reference

xlua.Export(function, func_name, [allowed_states...])
xlua.ClearExport(func_name)
xlua.Call(state_id, func_name, [args...])
xlua.Load(state_id, source, [name])
xlua.LoadFile(state_id, file)
xlua.Grant(feature, to_states...)
xlua.Revoke(feature, from_states...)
xlua.NewState(state_id)
xlua.DeleteState(state_id)
