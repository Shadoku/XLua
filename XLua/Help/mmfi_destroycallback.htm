MMFI: Destroy Callback

Description:

void mmf.DestroyCallback (number objectID)

This is a user-defined callback function. By default, it does not exist. If you define this function, it will be executed at the end of the event loop for every exported object that was destroyed in that iteration. The object ID will be passed as a parameter.

Defining this function is optional, but may help with more complex object management.

Parameters:

objectId
ID of an exported object.

Return Values:

none
