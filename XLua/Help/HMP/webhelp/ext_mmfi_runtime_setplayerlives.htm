void mmf.runtime.setPlayerLives ( number player, number lives )

Changes a player's number of lives to a new value.

Parameters

player Player index (1 - 4)
lives New number of lives

Return Values

None.

Example

local collectedLife = true
if (collectedLife) then
  mmf.runtime.setPlayerLives(1, mmf.runtime.playerLives(1) + 1)
end
