Expressions MMF Functions
  • XLua » Expressions » MMF Functions

This group of expressions is applicable once an MMF function has been invoked, and allows access to any parameters passed to the MMF function by the Lua code.  The values will remain valid until the MMF function returns.

When the call stack contains multiple frames (recusive function calling between Lua and MMF), each MMF frame will have its own set of parameters available to it, even if it invokes another MMF function or calls a Lua function that invokes an MMF function.

Get Integer Parameter

Function: MF_NumericParam
Returns: Value

Returns one of the parameters passed in an MMF function call from Lua as an integer value.

Parameters

Value 1-based parameter index

Get Floating Parameter

Function: MF_FloatingParam
Returns: Value

Parameters
