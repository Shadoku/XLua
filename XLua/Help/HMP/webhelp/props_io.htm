Properties Standard I/O
  • XLua » Properties » Standard I/O

These properties deal with the implementation of Lua's standard output and error streams.

Error Events

Instructs how the XLua object should handle triggered error events. The Immediate option causes events to be executed immediately, and the Queued option causes events to be executed at the end of the event loop.

Print Events

Instructs how the XLua object should handle triggered print events. The Immediate option causes events to be executed immediately, and the Queued option causes events to be executed at the end of the event loop.

Report Backtraces

Instructs whether the XLua object should include a full backtrace of the call stack when reporting runtime errors.
