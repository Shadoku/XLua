Get Error Event Mode

Function: ErrEventMode$
Returns: String

Returns Immediate if the error event mode is immediate, or Queued if the error event mode is queued.

Parameters

None.

Get Print Event Mode

Function: PrintEventMode$
Returns: String

Returns Immediate if the print mode is immediate, or Queued if the print mode is queued.

Parameters

None.
