Call With Existing Parameters

Function: LuaFunction$
Returns: String

Calls a lua function mid-expression with any previously pushed parameters, and returns the first result as a string. Any previous return results are discarded.

Parameters

String Function name

Call With Numeric Parameter

Function: LuaFunctionNP$
Returns: String

Calls a lua function mid-expression with a single integer parameter, and returns the first result as a string. Any previous return results or pushed parameters are discarded.

Parameters

String Function name
Value Integer parameter

Call With String Parameter

Function: LuaFunctionSP$
Returns: String

Calls a lua function mid-expression with a single string parameter, and returns the first result as a string. Any previous return results or pushed parameters are discarded.

Parameters

String Function name
String String parameter
