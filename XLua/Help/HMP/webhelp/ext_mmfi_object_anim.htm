Animation Traits correspond to the Animation action and expression menus.  The MMFI is able to determine at runtime without additional help whether an object supports these traits, and will expose the corresponding fields and methods if it does.

Animation Fields

These fields can be read from or written to directly

number animFrame Object's current frame of animation (1-based)
number animSet

Object's current animation set (1-based)
