Get Error String

Function: ErrorString$
Returns: String

Returns the last error reported. This is best used with the On Error condition.

Parameters

None.

Get Print String

Function: PrintString$
Returns: String

Returns the last string returned from a lua print statement. This is best used with the On Print condition.

Parameters

None.
