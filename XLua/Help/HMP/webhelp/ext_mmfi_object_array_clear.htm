void object.clear ( )

Resets all values in the array, in all dimensions, to 0.  This method has no effect on text arrays.

Parameters

None.

Return Values

None.

Example

-- Assume we have a an array object "arr"
-- Set (X=3,Y=2,Z=1) to 50
arr.set(50, 3, 2, 1)
assert(arr.get(3, 2, 1) == 50)
 
-- Now clear the array
arr.clear()
assert(arr.get(3, 2, 1) == 0)
