Movement Traits correspond to the Movement and Direction action and expression menus exposed by some objects, including the Active Objects.  The MMFI is able to determine at runtime without additional help whether an object supports these traits, and will expose the corresponding fields and methods if it does.

Since the MMFI is usually used in an environment of static movement, there is little support for directly interacting with MMF movements.

Movement Fields

These fields can be read from or written to directly

number direction Object's current direction value (1 - 32)
