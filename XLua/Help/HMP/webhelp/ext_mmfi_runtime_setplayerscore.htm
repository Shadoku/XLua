void mmf.runtime.setPlayerScore ( number player, number score )

Changes a player's score to a new value.

Parameters

player Player index (1 - 4)
score New score

Return Values

None.

Example

local collectedBonus = true
if (collectedBonus) then
  mmf.runtime.setPlayerScore(1, mmf.runtime.playerScore(1) + 1000)
end
