  • XLua » Extensions » Static Array

void array:reset ( )

Efficiently sets all elements of the array to 0.

Parameters

None.

Return Values

None.

Example

-- Assume we have an array "arr"
arr:reset()
 
-- Further actions have no effect
for i = 1, #arr do
  assert(arr[i] == 0)
end
