XLua: Load

Description:

boolean xlua.Load ( number stateID, string source, [string name] )

Loads a string containing Lua code into a remote state and executes it. If the call succeeds, it will return true.

A state can only load Lua code into a target state if the target state has granted the REMOTE_LOAD permission to the calling state.

Parameters:

stateID
The remote state in which to load Lua code
name
A string containing Lua code
name
An optional identifier for the chunk of code

Return Values:

boolean
true if the call succeeds, false otherwise
