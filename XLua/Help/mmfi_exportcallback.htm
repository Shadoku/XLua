MMFI: Export Callback

Description:

void mmf.ExportCallback (number objectID)

This is a user-defined callback function. By default, it does not exist. If you define this function, it will be executed every time you export an object. The object ID will be passed as a parameter.

Defining this function is optional, but may help with more complex object management.

Parameters:

objectId
ID of an exported object.

Return Values:

none
