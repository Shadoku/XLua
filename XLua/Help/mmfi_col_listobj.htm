MMFI: Get Objects Over Object

Description:

table mmf.Collisions.GetObjectsOverObject ( number objid [, number classid [, number layer [, number xoff [, number yoff] ] ] ] )

Returns a table containing the fixed IDs of all objects overlapping a given object. If a class ID is specified, only objects belonging to that class will be returned. If layer is specified, only that layer will be tested. If an X-offset or Y-offset are specified, they will be added to the second object's coordinates before testing.

The layer parameter may additionally be either of the special constants mmf.SAME_LAYER or mmf.DIFF_LAYER.

To get an object's class ID, use mmf.Object.GetClass.

To get an object from a fixed ID, use mmf.Object.Export.

Parameters:

objid
ID of object to use as collision mask
classid
ID of object class to test for
layer
1-based layer to test on
xoff
Distance in pixels to offset object's mask on the X axis
yoff
Distance in pixels to offset object's mask on the Y axis

Return Values:

boolean
table of fixed IDs
