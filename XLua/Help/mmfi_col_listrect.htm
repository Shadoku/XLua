MMFI: Get Objects Over Rect

Description:

table mmf.Collisions.GetObjectsOverRect ( number x, number y, number w, number h [, number classid [, number layer] ] )

Returns a table containing the fixed IDs of all objects overlapping a given rectangular region in the frame. If a class ID is specified, only objects belonging to that class will be returned. If layer is specified, only that layer will be tested.

To get an object's class ID, use mmf.Object.GetClass.

To get an object from a fixed ID, use mmf.Object.Export.

Parameters:

x
X (Left) coordinate
y
Y (Top) coordinate
w
Rectangle Width
h
Rectangle Height
classid
ID of object class to test for
layer
1-based layer to test on

Return Values:

boolean
table of fixed IDs
