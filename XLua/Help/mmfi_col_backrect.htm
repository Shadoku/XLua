MMFI: Backdrop Test Rect

Description:

boolean mmf.Collisions.BackdropTestRect ( number x, number y, number w, number h [, number layer] )

Tests a rectangular region of the background collision mask for an obstacle. If layer is specified, only that layer will be tested.

Parameters:

x
X (Left) coordinate
y
Y (Top) coordinate
w
Rectangle Width
h
Rectangle Height
layer
1-based layer to test on

Return Values:

boolean
true if there is an obsticle, false otherwise
