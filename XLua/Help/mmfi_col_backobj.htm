MMFI: Backdrop Test Object

Description:

boolean mmf.Collisions.BackdropTestObject ( number objid [, number layer [, number xoff [, number yoff] ] ] )

Tests the intersection of the background collision mask against the collision mask of another object for an obstacle. If layer is specified, only that layer will be tested. If an X-offset or Y-offset are specified, they will be added to the object's coordinates before testing.

The layer parameter may additionally be either of the special constants mmf.SAME_LAYER or mmf.DIFF_LAYER.

Parameters:

objid
ID of object to use as a collision mask
layer
1-based layer to test on
xoff
Distance in pixels to offset object's mask on the X axis
yoff
Distance in pixels to offset object's mask on the Y axis

Return Values:

boolean
true if there is an obsticle, false otherwise
