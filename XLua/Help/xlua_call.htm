XLua: Call

Description:

... xlua.Call ( number stateID, string name, [mixed arguments ...] )

Remotely calls a function in the specified state if it was previously exported with the specified name, and if the calling state was granted access to call the function.

A function may be called with any number of additional parameters. Valid parameters include booleans, numbers, strings, and tables. Tables will be deep-copied, with all internal references preserved (including cyclic references). Be aware that passing large tables will induce large overhead for copying the data. A function may return any number of return values, following the same rule as parameters.

All function calls are executed in the remote state, and affect only the remote state.

Parameters:

stateID
The remote state in which to call the function
name
The identifier of the function to call
arguments
Zero or more arguments to pass by value to the remote function

Return Values:

...
Zero or more return values returned by value
