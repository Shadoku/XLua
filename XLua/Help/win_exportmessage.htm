WINI: Export Message

Description:

void win.ExportMessage ( number message, function func )

Binds the specified message to the specified lua function. Message identifiers are predefined by Microsoft in the Windows API; refer to MSDN or other documentation for the values and meanings.

When the specified message is triggered, the associated function will be executed with the message's two parameters: WPARAM and LPARAM. Refer to MSDN or other documentation for the meaning of these parameters.

Parameters:

message
An integer message identifier
value
A predefined lua function

Return Values:

None

Example:

-- WM_ENTERMENULOOP is a message that is triggered when an application or context menu is opened.

menuOpen = false
local WM_ENTERMENULOOP = 0x211

function OnMenuOpen (wparam, lparam)
  menuOpen = true
end

win.ExportMessage(WM_ENTERMENULOOP, OnMenuOpen)
