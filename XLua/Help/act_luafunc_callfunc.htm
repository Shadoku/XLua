Call Lua Function

Suspends the current event and executes a lua function. All parameters pushed since the last function call will be passed as function parameters.

Any valid non-local function can be called, including functions in subtables or using the colon operator. Example function names are:

  • dosomething
  • do.something
  • do:something
  • a.b.c:d
