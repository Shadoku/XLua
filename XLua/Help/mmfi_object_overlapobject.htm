MMFI: Object Overlaps Object

Description:

boolean mmf.Object.OverlapsObject ( number objectID, number targetObjectID [, number layerMode [, number xOffset [, number yOffset] ] ] )

Tests if the current object is overlapping any part of a second target object. This function only tests overlap between two specific object instances; it does not test for overlap between an object and an object class.

An optional third parameter controls whether the overlapping objects can be on different layers.

An optional fourth and fifth parameter can perform the overlap test as if the first object had been offset in the X and Y directions.

Parameters:

objectID
ID of an exported object
targetObjectID
ID of an exported object to test overlap with
layerMode
An optional value that takes one of two constants from the mmf namespace: mmf.SAME_LAYER or mmf.ANY_LAYER. If mmf.ANY_LAYER is specified, the overlap test will ignore layers. If mmf.SAME_LAYER is specified, the test will fail if the two objects are on separate layers. mmf.SAME_LAYER is the default value.
xOffset
An optional value that is added to the first object's X coordinate prior to the overlap test. The object's coordinate is not actually changed.
yOffset
An optional value that is added to the first object's Y coordinate prior to the overlap test. The object's coordinate is not actually changed.

Return Values:

boolean
true if the object is overlapping the target object, false otherwise
