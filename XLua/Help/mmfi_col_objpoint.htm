MMFI: Object Test Point

Description:

boolean mmf.Collisions.ObjectTestPoint ( number objid, number x, number y [, number layer] )

Tests a single pixel of the frame for the presence of an object's collision mask. If layer is specified, only that layer will be tested.

Parameters:

objid
ID of object to test for
x
X coordinate
y
Y coordinate
layer
1-based layer to test on

Return Values:

boolean
true if the object overlaps the specified point, false otherwise
