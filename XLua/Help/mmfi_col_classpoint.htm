MMFI: Object Class Test Point

Description:

boolean mmf.Collisions.ObjectClassTestPoint ( number classid, number x, number y [, number layer] )

Tests a single pixel of the frame for the presence of a collision mask belonging to any object of the specified class. If layer is specified, only that layer will be tested.

To get an object's class ID, use mmf.Object.GetClass.

Parameters:

classid
ID of object class to test for
x
X coordinate
y
Y coordinate
layer
1-based layer to test on

Return Values:

boolean
true if an object of the specified class overlaps the specified point, false otherwise
