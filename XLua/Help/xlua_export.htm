XLua: Export

Description:

void xlua.Export ( function func, string name, [number allowedStates ...] )

Makes a function in the current state available to other states. Once a function is exported, other states will be able to remotely call the function.

The function parameter is the actual Lua function, whereas the name is the identifier that will be used by other states to call the function.

Any number of state IDs may be listed as optional parameters, in which case, only the listed states will be allowed to call this function. If no states are listed, then ANY state is allowed to call this function.

Once a function is exported, it will continue to persist even if all references to it become lost. If the function is unexported, it will then be allowed to die.

Parameters:

func
Lua function to export
name
The identifier of the function used by other states
allowedStates
Zero or more states allowed to call this function

Return Values:

None
